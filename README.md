### 1. CYP2C9 Overview
* **Gene Location:** Chromosome 10q23.33 (9 exons, 490 amino acids)
* **บทบาทหลัก:** การเผาผลาญในระยะ Phase I oxidative metabolism ของยากรดอ่อน (Weakly acidic drugs) รวมถึงยาที่มีดัชนีการรักษาแคบ (Narrow Therapeutic Index: NTI)
* **จำนวนอัลลีล:** มีมากกว่า 85 Star Alleles ในฐานข้อมูล PharmVar

#### สารตั้งต้น (Substrates), ตัวยับยั้ง (Inhibitors) และตัวกระตุ้น (Inducers) ของ CYP2C9
* **Major Substrates:** Warfarin (ยาต้านการแข็งตัวของเลือด), Phenytoin (~90% เผาผลาญผ่าน CYP2C9), Celecoxib (~95%), Ibuprofen (~75%), Diclofenac (~60%), Glipizide/Tolbutamide (>80%), Losartan, Siponimod (~79%)
* **Strong Inhibitors (เพิ่ม AUC >= 5 เท่า):** Fluconazole (ยารา; ยับยั้งรุนแรงที่สุด), Amiodarone (ยาต้านหัวใจเต้นผิดจังหวะ), Sulfaphenazole (ใช้เป็น selective probe ในห้องปฏิบัติการ), Miconazole
* **Moderate Inhibitors:** Fluoxetine, Fluvoxamine, Metronidazole, Voriconazole
* **Strong Inducers (ลด AUC >= 50%):** Rifampicin (กระตุ้นแรงที่สุด), Carbamazepine, Phenobarbital, St. John's Wort (ผ่านกลไก PXR-mediated induction)

### 2. CYP3A4 & CYP3A5 Overview
* **Gene Location:** Chromosome 7q21.1 (อยู่รวมกันเป็น Cluster บนโครโมโซม)
* **CYP3A4:** มีการแสดงออกในทุกคน (Constitutive expression) รับผิดชอบการเผาผลาญยาประมาณ 50% ของยาทั้งหมดในตลาด ทั้งในตับและลำไส้เล็ก
* **CYP3A5:** มีการแสดงออกในรูปแบบ Polymorphic expression โดยจะสร้างโปรตีนที่ทำงานได้สมบูรณ์ก็ต่อเมื่อมีอัลลีล ***1** เท่านั้น (พบในประชากรแอฟริกันสูงถึง ~70% แต่พบในยุโรปเพียง ~5-15%) ส่วนอัลลีล ***3** (rs776746) มีความบกพร่องในการตัดแต่งสาย RNA (Intron 3 splice defect) ทำให้ไม่สามารถสร้างโปรตีนที่ทำงานได้ (Non-expressor)

#### เจาะลึก (Deep Dive): บทบาทของ CYP3A5 ต่อการปลูกถ่ายอวัยวะและยา Tacrolimus
Tacrolimus (FK506) เป็นยาในกลุ่ม Calcineurin Inhibitor ที่มีดัชนีการรักษาแคบ (Narrow Therapeutic Index) มีความแปรปรวนของการตอบสนองในแต่ละบุคคลสูงมาก ซึ่งขึ้นอยู่กับฟีโนไทป์การแสดงออกของ CYP3A5 อย่างชัดเจน:
* **CYP3A5 Expressors (*1/*1 หรือ *1/*3):** เผาผลาญยาเร็วมาก ทำให้ระดับยาในเลือดต่ำกว่าเป้าหมาย เสี่ยงต่อการเกิดภาวะปฏิเสธอวัยวะอย่างเฉียบพลัน (Acute Rejection) จำเป็นต้องได้รับขนาดยาเริ่มต้นสูงกว่าปกติ 1.5-2 เท่า
* **CYP3A5 Non-Expressors (*3/*3):** เผาผลาญยาช้ามาก ทำให้เกิดการสะสมของยาในกระแสเลือด เสี่ยงต่อการเกิดพิษต่อไต (Nephrotoxicity), พิษต่อระบบประสาท (Neurotoxicity) และโรคเบาหวานชนิดใหม่หลังการปลูกถ่ายอวัยวะ (New-Onset Diabetes After Transplant: NODAT) เนื่องจากยาเข้าไปทำลายเซลล์เบต้าของตับอ่อน
